# College Exam Seat Planner

## Project Description
A web application to allocate classrooms for college and school exams. It uses minimum number of rooms and use fastly lower floor classrooms.

## Features
- Add new classrooms with details (room number, capacity, floor number, washroom availablity)
- View all classrooms which is added
- Allocate exam seats for students
- Shows allocation results with room details

## How to Use
1. Add classrooms by giving details
2. Enter number of students
3. Click Allocate Seats button
4. View the allocated all rooms

## Technologies Used
- HTML
- CSS
- JavaScript
